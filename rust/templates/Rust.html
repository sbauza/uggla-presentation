What is RUST ?

  • New systems programming language
  • Multi paradigm language (imperative, functional, object oriented (not fully))
  • By Mozilla and the Community
  • Dual license MIT and Apache v2.0
  • First stable release May 15th, 2015
  • Rust 2018 (v1.31), a major new edition released December 6th, 2018

Goal ?

Provide an alternative to C/C++ and also higher-level languages

Why RUST is exciting to me ?

And probably why you should look at it ?

Promises

  • Focus on safety--> avoid errors and help finding them !
  • Especially safe concurrency
  • Fast, really fast !

But also

  • Compiled language, few runtime dependencies
  • Produce binaries, that could be static with musl
  • Can interact easily with other languages via ABI and bindings (Python, C/C++, JS...)
  • State of mind, hard at compile time, but safe at runtime
  • Somewhere challenging !

Language features

  • Static type system with local type inference
  • Explicit mutability
  • Zero-cost abstractions
  • Runtime-independent concurrency safety
  • Errors are values
  • No null
  • Static automatic memory management
  • No garbage collection
